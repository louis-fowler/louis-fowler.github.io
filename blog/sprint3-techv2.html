Control flow & loops

In JavaScript the control flow is the order in which the computer executes code. For example in the code below, the code will be run in order from top to bottom. First setting the variable 'newNum' with the property of 10, then adding 5 to it, making newNum = 15, then subtracting 10 making newNum = 5.

let newNum = 10;

newNum += 5;

newNum -= 10;

A loop will just run over certain code until a criteria is met. Below is a simple for loop which runs the code 'newNum += 1', until newNum is 5;

for (let newNum = 0, newNum < 6, newNum ++)

For an everyday analogy, control flow is the order/routine of your day. Wake up, brush your teeth, eat breakfast, hop in your car...

Whilst a loop is repeating something until a desired outcome is achieved, brushing each tooth 5 times fro optimal cleaning

The DOM

The Document Object Model or DOM for short is an interface that interacts with HTML (and XML) to produce a node tree structure. It is an object-based representation of the source HTML.

Through the use of Devtools users are able to manipulate the DOM in there own browsers, changing CSS and HTML values such as text and style

Getting data from Arrays and objects

Although arrays and objects can look very similar, accessing the data inside of them is very different. Bellow is an example of a simple array and a simple object

let newArray = [1, 2, 3, 4, 5]



let newObject = {

    name: 'Henry',

    age: 27,

    gender: 'Male',

}

To access a certain number in the array we can pick it out with square bracket notation and its index in the array (index starts at 0 not 1!)

So to access number 3 from newArray we use newArray[4]

This can't be used for objects however, newObject[2] will return undefined! So how do we get the information out?

We can use either square bracket notation or dot notation. Say we went the name from newObject, to get it we can use newObject['name'] or newObject.name.

Functions, what are they?

Functions are a nice way to group a collection of code together and give it a name so we are able to run that collection whenever we want!

We are also able to pass a function new data every time (as long as it is applicable) to get different results! Below is an example that takes a number (x), multiplies it by 5 then subtracts 2 then console logs the new number.

const mathFunction = (x) => {

    x = x * 5 - 2,

    console.log(x)

}

We are able to pass the function a number via calling the function and giving it an argument -

mathFunction(5)

Which will console log the number 23!
