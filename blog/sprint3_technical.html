What is HTML & CSS

29/04/20

At a very basic and simple level, HTML (Hypertext Markup Language), is the bones of a webpage. This is where the content that is to be shown on the webpage is set. This includes setting titles, paragraphs of text, images, links to other pages etc.

CSS (Cascading Style Sheets) on the other hand is used to organise and style those bones. With CSS we can arrange HTML segments on the page. We can colour text and backgrounds. We can select fonts and the size fo text. We can do some very basic animation and much much more!

With all this talk of bones, lets use a skeleton as an analogy to describe the difference between HTML and CSS
