Margin vs Padding vs Border

21/4/20

Border

Above is a simple example of how an element is laid out on a page. The content in the middle, surrounded by padding, then border then finally margin.

Content

The content is the element in question. This is your image, text, div etc. Lets use a metaphor of an artwork, its frame and the wall it hangs on. For this metaphor consider the content as the actual artwork.

Border

A border is the edge of the content, imagine the frame on your painting.

Borders can be stylized through css with the 'border' property

With the border property we can style the width ( eg. 'border-width: 2px'), the colour ( eg. border-color: 'red') as well as the overall style, such as solid, dashed, doubled etc ( eg. border-style: 'dashed')

Padding

The padding property is used to add space around your contents borders, between the content and the border. Consider this an artworks matboard, the area between the artwork and the frame ( border )

Padding can be set through css with the 'padding' property

This can be done for individual sides with 'padding-top', 'padding-right', 'padding-bottom' and 'padding-left'. Or can be set all together with the shorthand property.

The shorthand is set in a clock wise rotation starting from the top

Example - padding: 10px 5px 15px 20px

in this instance the padding has been set as follows - padding-top: 10px, padding-right: 5px, padding-bottom: 15px and padding-left: 20px

Margin

Finally theres margin, this is where space OUTSIDE of the border is set. This helps sets the elements position inside its container. Consider this where on the wall you hang your painting.

Margins are set in css with the 'margin' property

Much like padding margins can be set individually with the margin-xxx property or all at one time with shorthand

The margin property can also be set to 'auto' (margin: 'auto') this centers the element horizontally within its container

but how does this look in action?

Example

fig. 1 boat.png on a webpage

fig. 2 boat.png with margin, padding and border highlights

In fig. 1 we can see an image element of a boat. This element has had its padding, border and margin set with css.

This particular image style is programed as so -

.imageContainer {

        padding: 50px;

        border: 5px dashed brown;

        margin: 80px;

}

We can see all the sections highlighted in different colours in fig. 2

Padding is set to 50px, with only one value set, each side inherits that value. All sides have a padding of 50px - This can be seen in fig .2 in green

border is set with shorthand to be 5px wide, have the dashed style and be brown. This can be seen in figs 1 & 2 as the brown dashed line

The margin is set once again using only one value, so all sides have the same margin size of 80px. This can been seen in fig. 2 in the orange outer section
